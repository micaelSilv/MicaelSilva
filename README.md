# Portfolio para GitHub Pages

Projeto estatico pronto para publicar no GitHub Pages.

## Arquivos principais

- `index.html`: estrutura do site
- `styles.css`: visual e responsividade
- `script.js`: cards de projeto, animacoes e ano automatico

## Como personalizar

1. Troque `Seu Nome`, textos e links em `index.html`.
2. Edite a lista `projects` em `script.js` para colocar seus projetos reais.
3. Ajuste cores e espacamentos em `styles.css` se quiser mudar a identidade visual.

## Como publicar no GitHub Pages

1. Crie um repositorio no GitHub.
2. Envie estes arquivos para a branch `main`.
3. No GitHub, abra `Settings > Pages`.
4. Em `Build and deployment`, escolha `Deploy from a branch`.
5. Selecione a branch `main` e a pasta `/ (root)`.
6. Salve e aguarde o link publico ser gerado.

## URL esperada

Se o repositorio se chamar `portfolio`, a URL normalmente fica assim:

`https://seuusuario.github.io/portfolio/`
